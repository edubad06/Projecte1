# Proyecto 1 - ControlCAT
## Autores: Violeta Bezunartea, Jordi Tijeras y Eduardo Badoyan

Nuestro proyecto, ControlCAT, tiene como objetivo diseñar e implementar un sistema para el control de asistencia del alumnado, profesorado y resto de personal del Institut TIC de Barcelona, que permita automatizar tanto la recogida de los datos de acceso al centro como su posterior consulta y gestión.

ControlCAT se basa en un dispositivo de lector de tarjetas de proximidad, que valida en tiempo real la identidad del usuario. El dispositivo envía la información hacia un servidor, donde es procesada y almacenada en la base de datos.

Además, el sistema incluye el diseño de una aplicación web y una aplicación móvil orientadas a cubrir las necesidades de los distintos usuarios. Dependiendo de cada uno de ellos, se podrá consultar asistencias, gestionar faltas y administrar usuarios entre otras funciones.

El resultado es un sistema completo que cubre la recogida, envío, almacenamiento, gestión y visualización de datos de asistencia.
